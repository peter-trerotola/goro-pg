go-postgres-mcp

read-only postgresql mcp server with schema intelligence

install
curl -sfL https://peter-trerotola.github.io/go-postgres-mcp/install.sh | sh

or

go install github.com/peter-trerotola/go-postgres-mcp/cmd@latest

or

docker pull ghcr.io/peter-trerotola/go-postgres-mcp:latest

linux/darwin · amd64/arm64

features

9 mcp tools

query, explore schemas, search tables, discover database structure

read-only

4 layers: ast parsing, ro connections, ro transactions, pg grants

schema intelligence

auto-discovery with sqlite fts5 knowledge map, schema context injection

works everywhere

claude desktop, claude code, cursor, any mcp client. stdio transport
