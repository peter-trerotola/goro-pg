go-postgres-mcp

Read-only PostgreSQL MCP server with schema intelligence

Install

curl -sfL https://peter-trerotola.github.io/go-postgres-mcp/install.sh | sh
or
go install github.com/peter-trerotola/go-postgres-mcp/cmd@latest
or
docker pull ghcr.io/peter-trerotola/go-postgres-mcp:latest

Supports Linux and macOS on amd64 and arm64.

9 MCP Tools

Query, explore schemas, search tables, and discover database structure through natural language.

Read-Only by Design

4 layers of protection: AST parsing, read-only connections, read-only transactions, and PostgreSQL grants.

Schema Intelligence

Auto-discovers schemas with SQLite-backed FTS5 knowledge map. Injects schema context into query responses.

Works Everywhere

Claude Desktop, Claude Code, Cursor, and any MCP-compatible client. Runs as a stdio server.
